# Chess Swiss Tournament Manager

Built with Next.js, TypeScript, Tailwind CSS, and shadcn/ui.

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.
